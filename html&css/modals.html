How to create a segment ?

With segments, you can send notifications to specific groups of users. With filters you can create a user segments. User Segment will give user numbers, if users pass those filters.

The filter can be embedded in following format:
For onclick feedify.tag(['sendTags', {'tag1':'value1','tag2':'value2', … }]); For page load function _feedifyTagReady(){ feedify.tag(['sendTags', {'tag1':'value1','tag2':'value2', … }]);}

Example: Creating a User Segment which gives users who added a product to cart but had not successfully completed checkout process.

Create a Segment with Filter tag “Add to Cart” with value equal to “true” AND a Filter tag “Checkout” with value equal to “false”
The below add to cart tag should be placed on Add to Cart button
Add to Cart Tag: feedify.tag(['sendTags', {'Add to Cart':'true','Checkout':'false'}]);
The below Checkout Tag should be placed at your Success page
Checkout Tag: feedify.tag(['sendTags', {'Add to Cart':'false','Checkout':'true'}]);

You can create unlimited segments like for Login, Product view, Page view, Added to cart, Shopping cart, Initiated checkout, Success page, Last active visitors, Last Purchased amount, Last Purchased day, Country, Purchases amount greater than etc
